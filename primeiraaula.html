Adobe DreamWeaver

O que é e para que serve o DreamWeaver?

É um programa que pode ser usado para criar páginas na web.

Podemos criar ou alterar códigos fontes de páginas da web.

Como trabalhamos ou fazemos?

Temos duas opções de usarmos o DremWeaver, podemos criar os conteúdos dos sites incorporando os elementos por itens nos
menus (comandos), ou criarmos o site programando (TAGs) com códigos em HTML, CSS ou outras linguagens de programação web.
